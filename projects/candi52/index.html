Realization of an application to manage internship requests for the firemen of the Haute-Marne.

AlrikRr PHP, Bootstrap,PHPMyAdmin

  • Twitter

  • Reddit

  • LinkedIn

Candi52

I was lucky to be able to complete a project that I started during a 4-week internship in a technical college.

The realization of this application will have lasted 10 weeks. (2 internships + summer job).

Results

Realization of an application to manage internship requests for the firemen of the Haute-Marne.
The main objective was to reduce paper consumption for internship registration.

I’m not going to lie to you, I was handed a sheet of paper (application for an internship) and I was asked to find a simpler and less restrictive solution.

After a brain-stroming of a few days, I went on a web application.

I coded the application in PHP/Bootstrap/JS on a Debian 9 server.
For the management of the firefighter’s personal accounts, I linked the authentication form to the SDIS Active Directory.
This way I had no secrets to store on the application and users had one and the same account.

Initially, Candi52 was in the intranet test phase for 5 months.
Without any worries, the application is now accessible online.
Entirely realized by myself. (database, Front-End, Back-End, Server Configuration, training on the application)

Features

  • Creating an account and modifying personal information
  • Importing courses from an Excel format into the database
  • Complete modification of internships
  • Creation of an application for an internship with complete follow-up (modification, validation, etc.)
  • Creation of three levels of access (Firefighter, Head of Center, Departmental School)
