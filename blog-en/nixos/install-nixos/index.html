Install for beginners - LUKS, EFI & BTRFS

June 19, 2020 AlrikRr

3 minute read

  • Twitter

  • Reddit

  • LinkedIn

Special thanks to Rayan Lahfa

ISO & VBOX configuration

This tutorial was realized with a virtual environment using VirtualBox, don’t forget to activate the EFI option into your VM configuration.

You can download the official NIXOS ISO here : nixos.org/download
Don’t hesitate to read the wiki here : nixos.org

Installation

If you have any issue, Google is your friend

For this install, I used a Virtual Disk of 25GiB.

# Display NixOS Wiki 
$ nixos-help  

# If you need to change the keyboard layout, use this command with you contry code
# here I used French layout
$ loadkeys fr

# Switch with root
$ sudo -i

# Display disks name and partitions
$ lsblk  

# Open  parted tool and free 2GB for the SWAP  
$ parted /dev/sda -- mklabel gpt  
$ parted /dev/sda -- mkpart primary 512MiB -2GiB  
$ parted /dev/sda -- mkpart primary -2GiB 100%  

# Create boot ESP
$ parted /dev/sda -- mkpart ESP fat32 1MiB 512MiB  
$ parted /dev/sda -- set 3 boot on  

# Encryption of the root partition  
$ cryptsetup luksFormat /dev/sda1
$ cryptsetup luksOpen /dev/sda1 nixos-root

# Check if you nixos-root is open
$ ls -l /dev/mapper

# Format the nixos-root into BTRFS
mkfs.btrfs -L nixroot /dev/mapper/nixos-root

# Create the SWAP partition
$ mkswap /dev/sda2
$ swapon /dev/sda2

# Check if the SWAP is working
$ free -hm

# Format the BOOT partition with vfat
$ mkfs.vfat -n BOOT /dev/sda3

# Mount nixos-root into /mnt
$ mount -t btrfs /dev/mapper/nixos-root /mnt

# Create btrfs subvolume, the top level will be nios
$ btrfs subvolume create /mnt/nixos
$ btrfs subvolume list /mnt

# umount /mnt and mount again nixos-root but this time using btrfs options
$ umount /mnt
$ mount -t btrfs -o subvol=nixos /dev/mapper/nixos-root /mnt
$ cd /mnt
$ ls

# Create btrfs subvolume, this part replaces LVM
$ btrfs subvolume create /mnt/home
$ btrfs subvolume create /mnt/var
$ btrfs subvolume create /mnt/nix
$ btrfs subvolume list /mnt
$ mkdir /mnt/boot

# Mount BOOT into /mnt/boot
$ mount /dev/sda3 /mnt/boot
$ ls -l /mnt

# Generate all the default configuration files for nixos
$ nixos-generate-config --root /mnt

# Start the install and enter root password
$ nixos-install

Basic commands

Hare are some basic commands for surviving on NixOS :

# Rebuild the OS, this command will take a "snapshot" of all your nixos files.
# This way in cas of issues on a nixos configuration, you will be able to 
# go back to a previous build.
$ nixos-rebuild switch

# Rebuilt and display errors
$ nixos-rebuild switch --show-case

# Search a nixos packet
$ nix search <name>

# Install a nixos packet
$ nix-env -iA nixos.<name>

All the magic of NixOS lies in its configuration.nix and the *.nix files you’ll be adding to it !
Indeed, configuration you NixOS via /etc/nixos/configuration.nix.
Once done, you can copy/paste your configuration.nix file to another freshly installed NixOS to automatically configure it with your settings !
